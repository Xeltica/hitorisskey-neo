# Hitorisskey Server

ひとりすきーサーバーおよび、Web UI。

## ビルド

```
# バックエンド

cd ./packages/backend
yarn install
yarn build

# フロントエンド
cd ./packages/frontend
yarn install
yarn build
```

## ライセンス

AGPL 3.0
